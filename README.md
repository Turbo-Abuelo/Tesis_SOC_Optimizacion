# Tesis: Optimización con Autoorganización Crítica (SOC)

Este repositorio contiene el código y los resultados de las simulaciones para mi tesis de [SU TIPO DE TESIS, ej. Maestría/Grado] sobre la aplicación de la búsqueda de Autoorganización Crítica (SOC Search) en problemas de optimización combinatoria.

**Problemas Investigados:**
1.  Modelo de Spin Glass de Ising
2.  Problema de Coloreado de Grafos

## Contenido del Repositorio

* `ising_soc_simulation.py`: Implementación de los algoritmos de búsqueda para el modelo de Ising.
* `coloring_soc_simulation.py`: Implementación de los algoritmos de búsqueda para el problema de Coloreado de Grafos.
* `notebooks/`: Contiene Jupyter Notebooks (`.ipynb`) para análisis interactivo y visualización de resultados.
* `experiments/`: Almacena los datos de las corridas de simulación más grandes o específicas.

## Cómo Usar

1.  **Clonar el repositorio:** `git clone https://aws.amazon.com/es/what-is/repo/`
2.  **Instalar dependencias:** `pip install -r requirements.txt` (si crea uno, ver abajo)
3.  **Ejecutar simulaciones:** Puede ejecutar los scripts `.py` directamente desde la terminal o a través de los Jupyter Notebooks.

Para más detalles, consulte los Jupyter Notebooks en la carpeta `notebooks/`.
